💭 느낀 점

C#과 문법적으로 비슷한 부분이 많아 익숙했지만,
자바스크립트는 자동으로 형변환이 이루어지는 점이 인상적이었다.

특히 같은 연산이라도 상황에 따라 결과가 달라지는 부분이 흥미로웠고,
동시에 예측하기 어렵다는 점에서 독특한 언어라고 느껴졌다.

개인적으로는 변수 선언 시 타입을 명확히 지정하는 방식이 더 안정적이라고 느꼈으며,
이러한 점에서 자바스크립트의 유연성과 안정성 사이의 차이를 체감할 수 있었다.

## 🚀 배운 내용 활용 계획

이번 수업에서 배운 자바스크립트의 기본 개념들을 바탕으로  
앞으로는 단순한 문법 암기를 넘어서 실제 코드에 적용하는 연습을 할 계획이다.

특히 ES6의 `let`, `const` 사용을 습관화하여 변수의 역할을 명확하게 구분하고,  
형변환이 자동으로 이루어지는 자바스크립트의 특성을 이해하여  
예상치 못한 오류를 줄이는 방향으로 코드를 작성하고자 한다.

또한 비교 연산 시 `===`를 사용하는 습관을 들이고,  
조건문과 반복문을 활용하여 간단한 로직을 직접 구현해보는 연습을 지속할 것이다.

나아가, 오늘 배운 내용을 바탕으로 작은 프로젝트나 예제 코드를 작성하면서  
자바스크립트의 동작 방식을 더 깊이 이해하고 실무에 적용할 수 있도록 발전시킬 예정이다.
