## 🧠 학습 내용 정리

### 1. 자바스크립트의 특징
- 웹에서 동작하는 대표적인 프로그래밍 언어
- 동적 타입 언어로, 실행 시 타입이 결정됨

### 2. ES6 변수 선언
- `let` : 재할당 가능
- `const` : 재할당 불가능 (상수)

### 3. 형변환 (Type Coercion)
- 자바스크립트는 상황에 따라 자동으로 타입을 변환함
- 예:
  ```javascript
  '5' + 1  // "51"
  '5' - 1  // 4
