## 오늘 배운것

1. 클래스
- 객체 생성 설계도
- constructor: 객체 생성 시 1회 실행
- 메서드는 prototype에 저장되어 인스턴스끼리 공유

class Student {
  constructor(name, group) {
    this.name = name;
    this.group = group;
  }
  introduce() {
    console.log(this.name);
  }
}

--------------------------------------------------

2. 화살표 함수
- function 대신 => 사용
- 한 줄이면 return 생략 가능
- 객체 반환 시 ()로 감싸야 함

const f = x => x * x;
const g = () => 'hello';
const obj = () => ({ a: 1 });

--------------------------------------------------

3. this 차이 (중요)
- 일반 함수: 호출 시 this 결정
- 화살표 함수: 상위 스코프 this 사용

obj.method = function() { console.log(this); } // obj
obj.method = () => { console.log(this); } // 전역

정리:
- 객체 메서드 → 일반 함수 사용
- 콜백 함수 → 화살표 함수 사용

--------------------------------------------------

4. 이터러블 & 유사 배열
- 이터러블: for...of 가능 (배열, 문자열 등)
- 유사 배열: length + index만 있음 (for...of 불가)

for (const x of arr) {}

Array.from(arrayLike); // 유사 배열 → 배열 변환

--------------------------------------------------

5. Rest / Spread

Rest (모으기)
function f(a, ...rest) {}

Spread (펼치기)
Math.max(...arr);
const copy = [...arr];
const obj = {...o1, ...o2};

--------------------------------------------------

6. 복사 개념

let a = 10;
let b = a; // 값 복사

let arr1 = [1];
let arr2 = arr1; // 주소 공유

let arr3 = [...arr1]; // 새 배열 (얕은 복사)

정리:
- 기본형 → 값 복사
- 객체/배열 → 참조(주소) 복사
- spread → 새로운 객체 생성 (얕은 복사)

--------------------------------------------------

7. 구조분해 할당

배열 (순서 기준)
const [a, b] = [1, 2];
const [a, , b] = [1, 2, 3];
const [a, ...rest] = [1, 2, 3];

객체 (키 기준)
const { name, age } = user;
const { name: newName } = user;
const { a = 10 } = obj;
const { ...rest } = obj;

--------------------------------------------------

8. 함수 매개변수 구조분해

function f({ name, age }) {}

function f({ id, spec: { ram }, brand = '삼성' }) {}

- 필요한 값만 바로 사용
- 가독성 좋음
- 기본값 설정 가능

--------------------------------------------------

- 클래스 = 객체 설계도
- 메서드는 prototype 공유
- 화살표 함수 = this 없음 (상위 this 사용)
- 객체 메서드는 일반 함수 사용
- Rest = 모으기 / Spread = 펼치기
- 배열은 순서 / 객체는 키로 구조분해
- 배열/객체 비교는 값이 아니라 주소 비교
