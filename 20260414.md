## 오늘 배운것
📘 자바스크립트 DOM 수업일지

1. querySelector / querySelectorAll

* `querySelector`
  : CSS 선택자를 사용해서 **첫 번째 요소 하나만 선택**

```javascript
const el = document.querySelector('#list');
```

* `querySelectorAll`
  : CSS 선택자를 사용해서 **모든 요소 선택 (NodeList 반환)**

```javascript
const items = document.querySelectorAll('#list li');
```

* NodeList는 배열처럼 `forEach` 사용 가능

```javascript
items.forEach(item => {
  console.log(item.textContent);
});
```

---

2. 리스트(list) 탐색하기

* `<ul>` 안의 `<li>` 요소 가져오기

```javascript
const items = document.querySelectorAll('#list li');
```

* 각각의 요소 접근

```javascript
items.forEach(item => {
  console.log(item.textContent);
});
```

---

3. id / class로 요소 찾기

* id로 찾기 (하나만 존재)

```javascript
const box = document.getElementById('box');
```

* class로 찾기 (여러 개)

```javascript
const texts = document.getElementsByClassName('text');
```

---

4. 요소 속성 변경하기

* 텍스트 변경

```javascript
element.textContent = '변경된 텍스트';
```

* 스타일 변경

```javascript
element.style.color = 'red';
element.style.backgroundColor = 'yellow';
```

* 클래스 추가 / 제거

```javascript
element.classList.add('active');
element.classList.remove('active');
```

---

5. 요소 생성 및 추가

* 새로운 요소 생성

```javascript
const li = document.createElement('li');
li.textContent = '새 항목';
```

* 부모 요소에 추가

```javascript
list.appendChild(li);
```

---

6. 요소 삭제

```javascript
element.remove();
```

---

📌 오늘 배운 핵심

* querySelector → 하나 선택
* querySelectorAll → 여러 개 선택
* DOM 요소를 선택해서 텍스트, 스타일, 클래스 변경 가능
* createElement로 요소 생성 후 appendChild로 추가
* remove로 요소 삭제 가능

## 느낀점
자바스크립트를 사용해서 웹 화면 구성 요소에 접근하고 수정하고 추가하는 방법에대해서 배웠다.

## 활용하기
동적으로 요소를 수정하고 추가 또는 삭제할수있다.
