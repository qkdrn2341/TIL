## 오늘 배운것
리액트와 JSX 
useState, props사용법

- React → UI를 만드는 자바스크립트 라이브러리
- 상태(State)가 바뀌면 화면(UI)이 자동으로 다시 렌더링됨

컴포넌트, 상태(State), JSX로 직접 DOM을 건드리지 않고 상태만 업데이트.

1. 컴포넌트 생성  
2. state로 데이터 관리  
3. props로 데이터 전달  
4. state 변경 → UI 자동 업데이트  



## 느낀점
DOM을 직접 수정하는 구조보다 훨씬 간결하고 효율적으로 코드를 짤 수 있어서 좋았다. 컴포넌트를 사용하면 UI를 재사용할수있다는 점이 마음에 들었다.


## 활용하기
props를 이용해 효율적으로 데이터를 전달하고 반복작업을 줄일수있다.
