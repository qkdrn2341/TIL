## 오늘 배운것
리액트와 JSX 
useState, props사용법

리액트는 값이 바뀌면 화면을 자동으로 다시 그려주는 도구 (자바스크립트 라이브러리)

const [text, setText] = useState("처음");

setText("변경");

컴포넌트, 상태(State), JSX로 직접 DOM을 건드리지 않고 상태만 바꾼다.



## 느낀점
DOM을 직접 수정하는 구조보다 훨씬 간결하고 
효율적으로 코드를 짤 수 있어서 좋았다.

## 활용하기
props를 이용해 효율적으로 데이터를 전달하고 반복작업을 줄이고 코드를 간결화 하자
